## Tech Stack
* **Frontend:** Flutter (Dart) for Android, iOS, and Web.
* **Architecture:** Provider/Riverpod for state management.
* **Backend Integration:** RESTful APIs / Firebase (Conceptual/Implementation details in report).

## Setup & Run Instructions
1. Ensure you have the [Flutter SDK](https://flutter.dev/docs/get-started/install) installed.
2. Clone this repository: `git clone <repo-url>`
3. Navigate to the project directory: `cd ShopEZ`
4. Install dependencies: `flutter pub get`
5. Run the app: `flutter run`
